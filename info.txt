John Myles White: Bandit Algorithms, O'Reilly 2013

Folders
-------
- cmd:  
contains non-compiled binaries, ie main applications (dev/prod) which should not be 
named 'main.go' but rather speak for themselves 
- pkg: packages
- bin: contains compiled binaries


Files
-----
- new.py: creates a "Hello Bandit!" go binary named 'main.go' in cwd (dev)
- main.go: current main package (dev)
- Makefile: compile dev main into bin directory, delete compiled binary, run tests
